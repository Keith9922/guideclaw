# 引路虾海报文案与设计说明

## 项目名称

**引路虾 GuideClaw**

## 一句话定位

**面向科研新人的研究导航系统**

## 主宣传语

**从陌生课题出发，快速走到文献梳理、缺口发现与方案生成。**

可替换版本：

- **让科研新人从“不会开始”，走到“有一版可讨论的研究方案”。**
- **不是简单论文问答，而是持续推进一个科研项目。**

## 项目介绍

引路虾 GuideClaw 是一个面向科研新人的研究导航系统。它通过 PI 编排与多 Agent 协作，围绕课题建立项目级知识库，完成文献梳理、研究缺口判断、方案生成与结果汇总。系统支持 PDF 上传、知识检索、持续追问和网页/PDF 双端交付，帮助科研新人更快理解一个陌生领域，并把模糊方向推进成可讨论、可执行的研究方案。

## 海报建议突出展示的核心问题

- 这个方向到底在研究什么？
- 当前已经做到哪一步？
- 什么值得研究？
- 我应该先看什么？
- 下一步该怎么推进？

## 海报建议突出展示的核心能力

### 1. 研究主题输入

- 输入一个陌生课题，快速开始研究导航
- 支持补充背景说明与上传 PDF

### 2. PI 编排任务

- PI 负责统筹研究焦点与下一步目标
- 统一调度多个 Agent 协作执行

### 3. 多 Agent 协作

- 文献助理
- 选题分析员
- 方案设计师
- 组会秘书

### 4. 项目级知识库

- 支持 PDF 上传
- 支持知识沉淀与检索
- 提供原始文献链接、DOI、引用信息

### 5. 最终交付

- 文献卡
- 缺口卡
- 方案卡
- 纪要卡
- 结果汇总网页
- 结果汇总 PDF

## 最终效果

**从“我不知道怎么开始”，到“我有一版可讨论、可推进的研究方案”。**

## 设计建议

### 整体风格

- 学术感
- 清晰
- 轻科技
- 研究工作台感
- 干净但不单调

### 配色建议

- 主色：深青绿 / 青蓝绿
- 辅色：白色、浅灰、淡青
- 强调色：亮一点的青绿色或湖蓝色
- 字体颜色：深灰黑，避免纯黑过重

### 版式建议

采用三段式海报结构：

#### 上半部分：品牌区

- Logo
- 项目名：引路虾 GuideClaw
- 一句话定位
- 主宣传语

#### 中间部分：能力流程区

建议流程：

**输入课题**  
→ **PI 编排任务**  
→ **多 Agent 协作**  
→ **知识库与证据**  
→ **结果汇总**

#### 下半部分：价值区

- 帮科研新人快速入门陌生领域
- 帮用户明确“该读什么、该问什么、下一步做什么”
- 把零散研究过程沉淀成结构化成果
